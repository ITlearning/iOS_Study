#### 하루 요약
오늘 굉장히 많은걸 해냈다. 프로필 사진을 직접 골라서 등록하는 것부터, 비록 아직 사진 찍는 단계까진 가지는 못했지만(시뮬레이션으로 돌리는거라 나중에 구현하는 것 같음) (이라고 영상에서 영어로 말하는 것 같았음..) 거의 근접했다. 또한 파이어 베이스라는 서버를 처음 가입하고 생성해 iOS앱에 직접 연동까지 해봤다. 실제로 계정을 생성하면 올라가는 것을 보고 상당히 재미있었다. 너무 흥미롭고 신기했다. 아마도 이런걸 하려고 앱을 공부하나 싶다. 클론 코딩의 참 뜻을 알아가는 듯 하다. 이전까지는 그냥그냥 너무 긴 코드라고 생각했던것들이 하나하나 뜯어서 코드를 읽을 수 있는 기회도 생기고 다양한 컨트롤러, 옵셔널, 기능들을 사용할 수 있어서 클론코딩 하나보다. 물론 이러고 내일, 내일 모레 가면 다 까먹겠지.. 이래서 꾸준히 해야하나보다 생각한다. 오늘도 수고 많았다 나 자신!
